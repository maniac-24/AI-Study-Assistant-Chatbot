# 🤖 AI Study Assistant Chatbot

An interactive AI chatbot built using Streamlit that helps students learn Computer Science topics like Python, DSA, OS, and DBMS.

---

## 🚀 Features

* 💬 ChatGPT-style conversation UI
* 🧠 Smart keyword-based responses
* ⚡ Typing animation for realistic interaction
* 🗑 Clear chat functionality
* 📚 Focused academic assistance

---

## 🛠️ Tech Stack

* Python
* Streamlit

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
