# 🤖 AI Study Assistant Chatbot

A smart chatbot built using Streamlit that helps students with Computer Science topics like Python, DSA, OS, and DBMS.

---

## 🚀 Features

* ChatGPT-style UI
* Typing animation
* Chat memory (session-based)
* Keyword-based intelligent responses
* Clear chat option

---

## 🛠️ Tech Stack

* Python
* Streamlit

---

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
